# Инструкция по публикации сайта WORLDCARD

## Вариант 1: GitHub Pages (Бесплатно, рекомендуется)

### Шаги:

1. **Создайте аккаунт на GitHub** (если нет):
   - Перейдите на https://github.com
   - Зарегистрируйтесь

2. **Создайте новый репозиторий**:
   - Нажмите "New repository"
   - Название: `worldcard-site` (или любое другое)
   - Выберите "Public"
   - Нажмите "Create repository"

3. **Загрузите файлы**:
   - Нажмите "uploading an existing file"
   - Перетащите файлы: `index.html` и `styles.css`
   - Нажмите "Commit changes"

4. **Включите GitHub Pages**:
   - Перейдите в Settings → Pages
   - В "Source" выберите "main" branch
   - Нажмите "Save"
   - Ваш сайт будет доступен по адресу: `https://ваш-username.github.io/worldcard-site/`

---

## Вариант 2: Netlify (Бесплатно, очень просто)

### Шаги:

1. **Перейдите на https://www.netlify.com**
   - Зарегистрируйтесь (можно через GitHub)

2. **Загрузите сайт**:
   - Перетащите папку с файлами в область "Drag and drop your site output folder here"
   - Или используйте "Add new site" → "Deploy manually"

3. **Получите адрес**:
   - Netlify автоматически даст вам адрес типа: `https://random-name-123.netlify.app`
   - Можно настроить свой домен в настройках

---

## Вариант 3: Vercel (Бесплатно)

### Шаги:

1. **Перейдите на https://vercel.com**
   - Зарегистрируйтесь

2. **Импортируйте проект**:
   - Нажмите "Add New Project"
   - Загрузите файлы или подключите GitHub

3. **Деплой**:
   - Vercel автоматически задеплоит сайт
   - Получите адрес типа: `https://your-project.vercel.app`

---

## Вариант 4: Платный хостинг (REG.RU, Timeweb и др.)

### Шаги:

1. **Купите хостинг и домен** (если нужно)

2. **Подключитесь по FTP**:
   - Используйте FileZilla или другой FTP-клиент
   - Данные для подключения получите у хостинг-провайдера

3. **Загрузите файлы**:
   - Загрузите `index.html` и `styles.css` в корневую папку (обычно `public_html` или `www`)

4. **Настройте домен**:
   - Привяжите домен к хостингу через панель управления

---

## Что нужно проверить перед публикацией:

✅ Все ссылки работают
✅ Изображения загружаются (если есть)
✅ Модальные окна открываются
✅ Сайт корректно отображается на мобильных устройствах
✅ Контактные данные актуальны

---

## Рекомендации:

1. **Для начала**: Используйте GitHub Pages или Netlify (бесплатно и просто)
2. **Для продакшена**: Рассмотрите платный хостинг с доменом
3. **SSL-сертификат**: Убедитесь, что сайт работает по HTTPS (большинство хостингов предоставляют бесплатно)
