# Подключение домена REG.RU к Vercel

## ✅ Если вы используете DNS-серверы Vercel (ns1.vercel-dns.com и ns2.vercel-dns.com)

**Это правильный подход!** Если вы настроили DNS-серверы Vercel в REG.RU, то Vercel сам управляет всеми DNS записями.

### Что нужно сделать:

1. **В REG.RU** (уже сделано ✅):
   - DNS-серверы: `ns1.vercel-dns.com` и `ns2.vercel-dns.com`

2. **В Vercel**:
   - Убедитесь, что домены `worldcard.online` и `www.worldcard.online` добавлены
   - Нажмите **"Refresh"** на каждом домене
   - Подождите 15-60 минут для распространения DNS

3. **Проверка**:
   - DNS-серверы должны распространиться
   - Статус в Vercel изменится с "Invalid Configuration" на "Valid Configuration"
   - Vercel автоматически создаст нужные DNS записи

---

## Шаг 1: Добавьте домен в Vercel

1. Перейдите в ваш проект на Vercel: https://vercel.com/dashboard
2. Откройте проект `worldcard`
3. Перейдите в **Settings** → **Domains**
4. В поле "Add Domain" введите ваш домен (например: `worldcard.ru` или `www.worldcard.ru`)
5. Нажмите **"Add"**

Vercel покажет вам DNS записи, которые нужно настроить.

---

## Шаг 2: Настройка DNS в REG.RU

### Вариант A: Подключение основного домена (например, worldcard.ru)

1. Войдите в панель управления REG.RU: https://www.reg.ru/user
2. Перейдите в **Домены** → выберите ваш домен
3. Откройте раздел **"DNS-серверы и зона"**
4. Выберите **"Использовать DNS-серверы REG.RU"** (если еще не выбрано)
5. Перейдите в **"Управление DNS-записями"**

6. **Добавьте/измените записи:**

   **Для основного домена (worldcard.ru):**
   - **Тип**: `A`
   - **Имя**: `@` (или оставьте пустым)
   - **Значение**: IP адрес от Vercel (обычно `76.76.21.21`)
   - **TTL**: `3600` (или автоматически)

   **Для поддомена www (www.worldcard.ru):**
   - **Тип**: `CNAME`
   - **Имя**: `www`
   - **Значение**: `cname.vercel-dns.com` (или значение, которое даст Vercel)
   - **TTL**: `3600`

### Вариант B: Только через CNAME (рекомендуется для поддоменов)

Если Vercel дал вам CNAME запись:
- **Тип**: `CNAME`
- **Имя**: `@` (для основного домена) или `www` (для поддомена)
- **Значение**: значение от Vercel (например, `cname.vercel-dns.com`)
- **TTL**: `3600`

**Важно**: В REG.RU для основного домена (@) может не работать CNAME. В этом случае используйте A-запись.

---

## Шаг 3: Ожидание распространения DNS

1. После настройки DNS записей в REG.RU, подождите **15-60 минут** для распространения изменений
2. В Vercel статус домена изменится на **"Valid Configuration"** когда DNS настроены правильно
3. Vercel автоматически выдаст SSL-сертификат (HTTPS)

---

## Шаг 4: Проверка

1. Проверьте DNS записи через онлайн-сервисы:
   - https://dnschecker.org
   - Введите ваш домен и проверьте A/CNAME записи

2. После распространения DNS:
   - Ваш сайт будет доступен по вашему домену
   - HTTPS будет работать автоматически

---

## Важные моменты:

✅ **DNS записи от Vercel**: 
   - Vercel покажет точные значения в настройках домена
   - Используйте те значения, которые указаны в Vercel

✅ **Время распространения DNS**: 
   - Обычно 15-60 минут
   - Может занять до 24 часов (редко)

✅ **SSL сертификат**: 
   - Vercel автоматически выдает бесплатный SSL
   - Работает только после правильной настройки DNS

✅ **WWW и без WWW**: 
   - Можно настроить оба варианта
   - В Vercel можно добавить оба домена и настроить редирект

---

## Если что-то не работает:

1. **Проверьте DNS записи** через dnschecker.org
2. **Убедитесь**, что в REG.RU записи сохранены правильно
3. **Подождите** еще немного (DNS может распространяться медленно)
4. **Проверьте в Vercel** статус домена в настройках

---

## Пример настроек в REG.RU:

### Для worldcard.ru (основной домен):
```
Тип: A
Имя: @
Значение: 76.76.21.21
TTL: 3600
```

### Для www.worldcard.ru:
```
Тип: CNAME
Имя: www
Значение: cname.vercel-dns.com
TTL: 3600
```

**Важно**: Используйте точные значения, которые даст вам Vercel в настройках домена!
