# Публикация сайта WORLDCARD на Vercel

## Быстрый способ (через веб-интерфейс)

### Шаг 1: Подготовка файлов
Убедитесь, что у вас есть:
- `index.html`
- `styles.css`

### Шаг 2: Загрузка на Vercel

1. **Перейдите на https://vercel.com**
   - Войдите в аккаунт (или зарегистрируйтесь)

2. **Создайте новый проект**:
   - Нажмите "Add New..." → "Project"
   - Или "Import Project"

3. **Загрузите файлы**:
   - **Вариант A**: Перетащите папку с файлами
   - **Вариант B**: Подключите GitHub репозиторий (если файлы там)

4. **Настройки проекта**:
   - **Framework Preset**: Other (или оставьте по умолчанию)
   - **Root Directory**: `./` (корневая папка)
   - **Build Command**: оставьте пустым (для статического сайта не нужен)
   - **Output Directory**: оставьте пустым или `./`

5. **Деплой**:
   - Нажмите "Deploy"
   - Дождитесь завершения (обычно 1-2 минуты)

6. **Получите адрес**:
   - Vercel автоматически даст адрес типа: `https://your-project.vercel.app`
   - Можно настроить свой домен в настройках проекта

## Способ через Vercel CLI (для продвинутых)

### Установка CLI:
```bash
npm i -g vercel
```

### Деплой:
```bash
cd /Users/dimasnegovik/Desktop/Сайт\ WORLDCARD
vercel
```

Следуйте инструкциям в терминале.

---

## Настройка кастомного домена (опционально)

1. В настройках проекта → **Domains**
2. Добавьте свой домен
3. Настройте DNS записи согласно инструкциям Vercel

---

## Важные моменты для статического сайта:

✅ **Framework Preset**: выберите "Other" или оставьте по умолчанию
✅ **Build Command**: оставьте пустым
✅ **Output Directory**: оставьте пустым
✅ Vercel автоматически определит статический сайт

---

## После деплоя:

- Сайт будет доступен по адресу `https://your-project.vercel.app`
- При каждом обновлении файлов можно задеплоить заново
- Vercel автоматически создает HTTPS сертификат
- Можно настроить автоматический деплой из GitHub
